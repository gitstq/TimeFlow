## [1.0.0] - 2025-06-01

### 🎉 初始发布

#### ✨ 核心功能

- **时间追踪** - 开始/停止会话追踪
- **项目管理** - 自动检测和手动管理项目
- **智能检测** - 从 Git 和配置文件自动识别项目
- **报告生成** - 日/周/月报告，支持 JSON/CSV 导出
- **TUI 界面** - 美观的终端交互界面
- **数据持久化** - 本地 SQLite 存储

#### 🛠️ 技术特性

- Python 3.9+ 支持
- 完整的类型注解
- 单元测试覆盖
- 代码格式化 (Black)
- 静态类型检查 (MyPy)
- 代码质量检查 (Ruff)

#### 📚 文档

- 多语言 README (简体中文、繁体中文、English)
- 贡献指南
- MIT 开源协议
